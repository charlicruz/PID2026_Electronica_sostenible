# Electrónica sostenible — PID UAH/EV1812

Web estática preparada para GitHub Pages del proyecto de innovación docente:

**Electrónica sostenible: recursos docentes sobre ecodiseño, modularidad y reutilización de plataformas electrónicas**

## Contenido

- `index.html`: página principal.
- `styles.css`: hoja de estilos responsive.
- `assets/logo.png`: logo del proyecto actualizado.
- `assets/memoria-PID-UAH-EV1812.pdf`: memoria del proyecto en PDF.

## Publicación en GitHub Pages

1. Crear un repositorio nuevo en GitHub, por ejemplo `pid-electronica-sostenible`.
2. Subir todos los archivos y carpetas de este paquete a la raíz del repositorio.
3. Ir a `Settings > Pages`.
4. En `Build and deployment`, elegir `Deploy from a branch`.
5. Seleccionar la rama `main` y la carpeta `/root`.
6. Guardar y esperar a que GitHub publique la web.

## Personalización

- Para cambiar el vídeo, modificar en `index.html` la URL del iframe:
  `https://www.youtube.com/embed/miIaRYKboC8`.
- Para cambiar el cuestionario, reemplazar el enlace de Microsoft Forms en los botones y en el pie de página.
- Para actualizar la memoria, sustituir `assets/memoria-PID-UAH-EV1812.pdf` manteniendo el mismo nombre de archivo.
